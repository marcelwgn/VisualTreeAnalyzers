Class VisualTreeWalker

The VisualTreeWalker can be used to scan entire subtrees of the VisualTree. Every analyzer added to the VisualTreeWalker will be called on all elements visited. The tree walker might not visit all nodes if no analyzer requests to visit the children of a specific element.

The VisualTreeWalker will always perform a breadth-first-search on the visual tree.
